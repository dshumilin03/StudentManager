# Student Manager System

## Entities

### Student
Student is an entity that has:
- id
- fullName
- groupNumber
- age
- email

### Group (a group of students)
Group includes a bunch of students, it has:
- id
- groupNumber

## Api possibilities

### Student

- Add student
- Get student
- Update student data
- Delete student

### Group

- Add group
- Get group
- Update group data
- Delete group
